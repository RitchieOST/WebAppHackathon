Beschreibung


Wasserkraft im Vergleich zu PV Erstelle ein Diagramm zum Verlauf des Niederschlags und der Globalstrahlung zu einer frei gewählten Messstation von SwissMetNet. Für eine bessere Aussage des Diagramms stelle einen kumulierten Verlauf dar, d.h. eine ansteigende Kurve übers Jahr, welche die Energiezunahme darstellt. Interpretiere das Diagramm hinsichtlich der Unterschiede der einzelnen Jahre. Zeige das Wasserkraftpotenzial pro Jahr durch eine Wahl des Einzugsgebiets (u.a. mit Bildern einer Karte) dar. Visualisierung des Einzugsgebiets KW-Samina mit den relevanten Wetterdaten.


Steps


1. Historische Niederschlags- und Strahlungswerte für den Graphen über URL abgefragt.

2. Flächenmass des Einzugsgebietes über Geoportal kombiniert.

3. Niederschlagsmenge über Gesamtfläche mit Fallhöhe ergibt Energie- menge im KW-Samina (verworfen)

4. Graph über matplotlib erzeugen und als PNG automatisch im "static"-Verz. abspeichern. => Übergabepunkt an Django

5. Live-Wetterkarte über API per Key abgefragt und in das HTML eingebunden

Lessons learned

1. GitHub: Push/Pull befehle am besten mit Absprache

2. Über API richtige Daten abfragen (Plausibilitätskontrolle)

3. Projekt von Anfang an in Django erstellen, falls Visualisierung gewünscht.

4. Ordnerstruktur einhalten. Files dort ablegen wo sie gebraucht werden

5. KI-generierte Codes probieren zu verstehen, sonst wird Debugging schwierig
